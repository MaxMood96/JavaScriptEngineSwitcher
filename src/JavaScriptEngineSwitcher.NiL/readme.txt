   DESCRIPTION

   NiL.JS (https://github.com/nilproject/NiL.JS) version 2.5.1684).

   =============
   RELEASE NOTES
   =============
   1. NiL.JS was updated to version 2.5.1684;
   2. Added support of .NET 8.0.
